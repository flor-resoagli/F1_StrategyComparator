# F1_StrategyComparator

Tool to help compare different Formula 1 Race Strategies for a specific circuit, based on a Machine Learning model trained on race data from the Formula 1 2022, 2023, 2024 and 2025 seasons.

## How to use

1. Build the machine learning model using the prepare_app.py file. Select whichever model is preferred and generate the model file (Model files are too large to push into github).

2. Run the Flask API on the api.py file. The application will run on localhost:5000.

3. Run the 'strategy-simulator' React App. It will open on localhost:3000.
